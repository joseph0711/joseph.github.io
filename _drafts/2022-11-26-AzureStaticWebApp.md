## 本篇重點

此內容為以純 Web（HTML）上架步驟，其他 Framework 後續會繼續撰寫。

## 準備事項

1. Azure 訂閱帳戶
2. Github 帳戶
3. Visual Studio Code

## 操作步驟

### 將專案上傳至 Github Repository

建立 repo 並將檔案（至少一個 html 檔）push 至 repo 中。亦可透過 VSCode 套件將其上傳並發佈。
